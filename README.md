## Description

Nest.js勉強中。まだ慣れてないのでたまに気が向くと更新されてるはず

## Installation

```bash
$ npm install
```

## Build for release

```bash
$ npm run build
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# test coverage
$ npm run test:cov
```
